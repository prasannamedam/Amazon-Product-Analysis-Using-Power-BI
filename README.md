# Amazon-Product-Analysis-Using-Power-BI
🛒 Amazon Product Sales & Customer Insights Dashboard

📖 Description

This project presents an interactive Power BI dashboard for analyzing Amazon product data. It provides insights into product performance, customer ratings, pricing, discounts, and reviews, helping businesses understand sales trends, customer preferences, and product performance through data visualization.

🎯 Goal

To analyze Amazon product data and create an interactive dashboard that enables businesses to monitor product performance, evaluate customer feedback, and support data-driven pricing and marketing decisions.

🛠️ Technologies Used

Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel / CSV

📂 Project Workflow

1. Data Collection
Imported the Amazon product dataset into Power BI.
Reviewed data structure and attributes.
2. Data Cleaning
Removed duplicate records.
Handled missing values.
Corrected data types.
Transformed data using Power Query.
3. Data Modeling
Built relationships between tables (if applicable).
Created calculated columns and DAX measures.
4. Dashboard Development

Developed an interactive dashboard with:

Product Category Analysis
Product Rating Analysis
Price & Discount Analysis
Customer Review Analysis
Top-Rated Products
Most Reviewed Products
Discount Distribution
Product Performance Summary

📊 Dashboard KPIs

📦 Total Products

⭐ Average Rating

💰 Average Price

🏷️ Average Discount Percentage

📝 Total Reviews

🔥 Highest Rated Product

📈 Dashboard Visualizations

KPI Cards :

Clustered Bar Chart
Column Chart
Donut Chart
Pie Chart
Treemap
Line Chart
Table / Matrix
Slicers for Interactive Filtering

🔍 Key Insights :

Identified the top-performing product categories based on customer ratings.
Analyzed pricing trends and discount strategies across products.
Identified products with the highest customer ratings and review counts.
Compared product prices across different categories.
Evaluated the relationship between discounts and customer ratings.
Provided interactive filtering for better business analysis.

🚀 Future Scope :

Integrate live Amazon product data using APIs.
Add sales forecasting and trend analysis.
Build customer recommendation insights.
Include regional sales and product performance analysis.
Publish the dashboard to the Power BI Service for real-time reporting.

✅ Conclusion :

This project demonstrates how Power BI can transform raw Amazon product data into meaningful business insights. Through interactive dashboards, KPI reporting, and data visualizations, it enables businesses to monitor product performance, understand customer behavior, and support data-driven decision-making.
